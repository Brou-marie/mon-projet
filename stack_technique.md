## Stack technique complète (projet AfriStay)


### Backend (Django)
Python 3.11+
Django 5.0 (configuration + logique serveur)
Django REST Framework (DRF) (API REST JSON)
djangorestframework-simplejwt (auth JWT: access/refresh)
psycopg2-binary (PostgreSQL)
django-cors-headers (CORS)
django-filter (filtres API)
Pillow (gestion/traitement d’images)
python-dotenv (variables d’environnement)
Celery + Redis (tâches asynchrones + cache/queue)
Whitenoise (static files en production)
Gunicorn (serveur WSGI en prod)


### Frontend (React):
Node.js 18+
React 18 (UI)
Vite (bundler/dev server)
React Router DOM v6 (routing)
Axios (client HTTP)
React Query (data fetching/cache)
Tailwind CSS (design system rapide)
lucide-react (icônes)
date-fns (manipulation dates)
clsx + tailwind-merge (gestion classes Tailwind)
